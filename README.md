# 🏭 Smart Conveyor Inspection System

An advanced simulation of an industrial conveyor-based inspection system using computer vision and real-time motion.

---

## 🚀 Features

- Moving conveyor belt simulation
- Rotating rollers
- Inspection machine
- Defect detection:
  - Scratch
  - Crack
  - Hole Defect
  - Missing Material
  - Dent
- Defect size measurement (mm)
- Defect position detection
- Confidence and severity
- PASS / REWORK / REJECT decision system
- Inspection history table
- CSV download

---

## 🛠️ Technologies Used

- Python
- Streamlit
- OpenCV
- NumPy
- Pandas

---

## ▶️ How to Run

pip install -r requirements.txt  
streamlit run app.py  

---

## 📌 Project Description

This project simulates a real industrial conveyor inspection system where products move on a conveyor, get inspected using computer vision, and are automatically classified based on defects.
